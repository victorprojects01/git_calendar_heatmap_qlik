# GitHub Calendar Heatmap — Qlik Sense Extension

## 🇺🇸 English

GitHub-style contribution calendar for Qlik Sense / Qlik Cloud.

### Features

* 1 Date Dimension
* 1 Measure
* Automatic color intensity
* Theme color support
* Tooltip with value and date
* Multi-day selection
* Optional legend
* Smooth animations
* Light mode by default

---

## How to Import into Qlik Cloud

### 1. Download the ZIP

Download the extension `.zip` file.

### 2. Open Qlik Cloud

Go to your Qlik Cloud tenant.

### 3. Open Management Console

Click:

* **Profile Icon**
* **Administration**

### 4. Open Extensions

Inside Administration:

* Go to **Extensions**

### 5. Upload the Extension

Click:

* **Add**
* Select the `.zip` file

### 6. Use in an App

Open your Qlik app:

* Edit Sheet
* Open **Custom Objects**
* Drag **GitHub Calendar Heatmap** into the sheet

### 7. Configure

Add:

* 1 Date Dimension
* 1 Measure

Done 🚀

## Como importar no Qlik Cloud

### 1. Baixe o ZIP

Baixe o arquivo `.zip` da extensão.

### 2. Abra o Qlik Cloud

Entre no seu tenant do Qlik Cloud.

### 3. Abra o Console de Administração

Clique em:

* **Ícone de Perfil**
* **Administração**

### 4. Vá em Extensões

Dentro da administração:

* Abra **Extensions**

### 5. Faça Upload da Extensão

Clique em:

* **Add**
* Selecione o arquivo `.zip`

### 6. Use na Aplicação

Abra sua aplicação Qlik:

* Editar Sheet
* Objetos Customizados
* Arraste o **GitHub Calendar Heatmap**

### 7. Configure

Adicione:

* 1 dimensão de data
* 1 medida
